# Coffee Shop

The website includes: a coffee shop homepage, drinks and pastry menu, category filter, product cards, order cart, table reservation, contact form, and a modern minimalist design.

Technologies: React, TypeScript, Vite, Tailwind CSS, Responsive Design.

Functionality: Browse the coffee shop menu, filter products by category, add products to the cart, change product quantity, place a takeaway order, reserve a workspace, send a message through the contact form, responsive UI.
